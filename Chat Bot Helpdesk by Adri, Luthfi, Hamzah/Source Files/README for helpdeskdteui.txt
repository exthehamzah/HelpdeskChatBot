[HOW TO USE CHAT BOT HELPDESK DTE FTUI RESOURCE FILE]

1. First, install the XAMPP application
XAMPP download link: https://www.apachefriends.org/download.html
How to install: https://www.wikihow.com/Install-XAMPP-for-Windows

2. Download the 'helpdeskdteui' folder from Google Drive

3. Move/copy the folder into XAMPP's htdocs folder
Location/directory: ...\xampp\htdocs\
*Note: "..." can be anything, so check your own XAMPP application directory

4. Open the "XAMPP Control Panel"

5. Turn on 'Apache' Module (just click 'Start' button in the Apache Module's row)

6. Open the internet browser

7. Go to address "localhost/'helpdeskdteui/"

8. Welcome to our semi-offline website (for using the chat bot, the internet connection is still required)
